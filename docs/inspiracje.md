# Inspiracje projektu

## I. Inspiracje (słowa, skojarzenia, pomysły)
- **minimalizm UI**
- automatyzacja rutynowych zadań
- personalizacja ustawień
- szybkie filtrowanie i wyszukiwanie
- integracja z kalendarzem / powiadomieniami
- responsywność i dostępność
- wizualne podsumowania (dashboard, wykresy)
- tryb offline / synchronizacja
- modułowe rozszerzenia (pluginy)

## II. Krótka lista elementów projektu
- interfejs użytkownika (dashboard, formularze, listy)
- baza danych (proste API lub lokalne przechowywanie)
- warstwa logiki (filtrowanie, reguły, harmonogram)
- moduł synchronizacji (opcjonalny)
- system powiadomień i przypomnień
- opcje eksportu/importu danych *(CSV, JSON)*
- panel ustawień i personalizacji

## III. Wstępne dane wejściowe
- lista kategorii: "Praca", "Prywatne", "Nauka", "Zakupy", "Inne"
- lista priorytetów: "Niski", "Średni", "Wysoki"
- funkcje dostępne: dodaj/edytuj/usuń element, oznacz jako wykonane, przypisz datę i kategorię, wyszukaj, filtruj po tagach
- parametry synchronizacji: częstotliwość (np. co 15 min), typ połączenia (Wi‑Fi/Komórkowe)
- ustawienia użytkownika: język, motyw (jasny/ciemny), powiadomienia włącz/wyłącz

## IV. Lista źródeł pomysłu
- **własna potrzeba:** usprawnienie codziennego planowania zadań
- **obserwacje znajomych:** potrzeba prostych narzędzi bez nadmiaru funkcji
- **istniejące aplikacje:** inspiracja układem dashboardu i filtrowaniem
- **przypadek:** brak dostępnych narzędzi do szybkiego eksportu prostych raportów
