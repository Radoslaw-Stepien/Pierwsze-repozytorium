# Opis projektu

1. Projekt dotyczy aplikacji pomagającej studentom planować naukę i zarządzać zadaniami.
2. Rozwiązuje problem braku struktury oraz przeładowania informacjami, dzięki czemu użytkownicy szybciej identyfikują priorytety i harmonogram nauki.
 
 **Odbiorcami** są osoby w wieku **18–25** lat, które chcą lepiej organizować zadania akademickie i czas wolny.

 W aplikacji znajdą się:
 - lista zadań,
 - kalendarz tygodniowy
 - oraz proste statystyki śledzące postępy i nawyki.
 
 Celem projektu jest zwiększenie efektywności nauki poprzez prosty interfejs, klarowną strukturę planowania i motywujące informacje zwrotne.
